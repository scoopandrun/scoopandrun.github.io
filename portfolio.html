Blog

Ce projet a été réalisé lors de ma formation chez OpenClassrooms. Le but était de créer un blog de A à Z, sans utiliser de CMS ou de framework PHP.

ToDo & Co

Ce projet a été réalisé lors de ma formation chez OpenClassrooms. Le but était de créer un gestionnaire de tâches.
L'exercice a commencé avec une application de base réalisée avec Symfony 3.1 et Bootstrap 3.3.7 qui devait être mise à jour aux dernières versions (Symfony 7.1 et Bootstrap 5.3 à l'époque).
En plus de la mise à jour des versions, un système d'autorisation et un panneau d'administration devaient être implémentés.
Une partie significative du projet était d'écrire des tests unitaires et fonctionnels pour toutes les fonctionnalités.
L'application a été réalisée avec PHP (Symfony), Bootstrap, et MySQL.

This project was made during my training at OpenClassrooms. The goal was to create a task manager.
The exercise started with a base application made with Symfony 3.1 and Bootstrap 3.3.7 that had to be updated to the latest versions (Symfony 7.1 and Bootstrap 5.3 at the time).
Along the version upgrade, an authorization system and admin panel had to be implemented.
A significant part of the project was to write unit and functional tests for all the features.
The application was made with PHP (Symfony), Bootstrap, and MySQL.
